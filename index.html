Frequently Asked Questions

Is my audio data secure and private?

Absolutely. Both tools run entirely in your browser using WebAudio API. Your audio never leaves your device unless you explicitly choose to export or share. No data is sent to our servers, ensuring complete privacy and security for your creative work.

Which browsers and operating systems are supported?

Best Performance: Chrome 66+, Edge 79+, Brave, Opera
Good Support: Firefox 60+
Limited Support: Safari 14.1+ (some features may be limited)
Works on Windows, macOS, Linux, Android, and iOS. For optimal performance, we recommend using a desktop browser with at least 4GB RAM.

Can I use this offline? Do I need to install anything?

Zero Installation Required. Both tools work directly in your browser with no plugins or downloads. You can add the website to your home screen (PWA) for offline access. Once loaded, the tools work completely offline, making them perfect for mobile music creation.

What audio equipment do I need for best results?

Minimum: Built-in microphone or audio input
Recommended: USB microphone or audio interface, closed-back headphones
Professional: ASIO-compatible audio interface for <5ms latency
Tips: Close other browser tabs, use wired connections, set sample rate to 44.1/48kHz in system settings.

What file formats are supported for audio import/export?

Import: WAV, MP3, OGG, M4A, FLAC
Export: WAV (uncompressed), MP3 (various bitrates)
LoopStation Pro: Export individual tracks or full stereo mix
Audio ‚Üí Synth: Export rendered performances, save synthesizer presets
Maximum file size: 50MB per import.

Is there a mobile version or app available?

Progressive Web App (PWA): Yes! Add the website to your phone's home screen for an app-like experience. Works on iOS Safari and Android Chrome with touch-optimized controls.
Performance Note: Mobile devices may have higher latency and fewer simultaneous tracks. Desktop/laptop recommended for professional use.
